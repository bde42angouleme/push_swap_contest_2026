## Comment participer
Avant de commencer, fork le projet avec le bouton fork au-dessus de la description du repo. Une fois que tu as fait cela, tu pourras faire tes modifications de ton repo personnel.

Lorsque tu as terminé, tu pourras créer une pull request. Pour ce faire, sur la page de ton repo, un bouton `contribute` apparaît avec lequel tu pourras créer une pull request. ✌️

Toutes les pull requests seront mergées à la fin du concours.

**Good luck :)**
